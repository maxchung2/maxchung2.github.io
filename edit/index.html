Foodpak

FoodPak is a family run business established in 1968 that provides custom-tailored packaging solutions for clients across the Pacific Northwest and beyond.

For FoodPak's new website, the client supplied some profesionally-taken product photos, however the background needs to be removed so it can be used on all backgrounds. Most of the backgrounds can be removed with colour to alpha, but some tricky ones such as the clamshell tray required a green screen plugin.
