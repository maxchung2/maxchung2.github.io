Logo Design

Tempus

Tempus is a web application to help health authorities track grievances and submit these reports to the Health Employers Association every quarter. The brand and name refresh aims to give the application a modern and approachable look with rounded duotone shapes.

The logo is a combination of symbols.

  1. The interaction between two unhappy people (faces)
  2. The protection of the organization (shield)
  3. The name Tempus, which means time in Latin (hourglass)

JetSweep Data Analytics

JetSweep is an AWS Infrastructure Cloud Services provider that delivers solutions to customers in the areas of Cloud Infrastructure, Cloud Migration, Back-Up & DR, End User Compute and DB & Analytics. We decided to create a new logo for them along with a website refresh.

The logo is a combination of symbols.

  1. The prevalence of Infrastructure as Code for configuration consistency (</xml> tag)
  2. The modular nature of cloud containers and instances (cube shape)
  3. The movement of network traffic (< and >)
  4. The letters J and S that makes up the name JetSweep
