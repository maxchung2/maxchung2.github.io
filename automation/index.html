Tempus Regression Testing

Tempus is a web application to help health authorities track grievances and submit these reports to the Health Employers Association every quarter.

To help detect bugs in the development of the Tempus web application, I wrote a Selenium Java automation program to test the functions on each page. As the program fills form fields and toggles filters, it checks if the results match the inputted data. If not, it logs the error and attempts to continue.
