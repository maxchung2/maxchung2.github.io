Tempus Select Dropdown Widget

Tempus is a web application to help health authorities track grievances and submit these reports to the Health Employers Association every quarter.

While testing the Tempus web application for bugs, I noticed adding a new grievor to a grievance required going through three levels of modals:

  • Add a grievor
  • Add a new grievor
  • Select cost centre

I decided to challenge myself to come up with a better solution and apply the solution to other cases. The 'Add a grievor' and 'Select cost centre' modals became dropdown menus and the 'Add a grievor' remains as a modal.
